Ein wenig verspätet, aber ich bin erst jetzt dazu gekommen mich damit zu beschäftigen.

Aber [groovy 2.3.0](http://docs.codehaus.org/display/GROOVY/2014/05/05/Groovy+2.3.0+is+out) ist raus! Yeah! Ich bin gespannt. 

Die [Releasenotes](http://groovy.codehaus.org/Groovy+2.3+release+notes) lesen sich wirklich sehr spannend :)

Das ist die erste Version die mit java 8 kompatibel ist.
Aber die zwei dicken Bretter, wie ich finde sind die Einführung von [traits](http://groovy.codehaus.org/Groovy+2.3+release+notes?nc#Groovy2.3releasenotes-Traits) und der [Perfomanceboost](http://groovy.codehaus.org/Groovy+2.3+release+notes?nc#Groovy2.3releasenotes-DrasticJSONparsingandserializationperformanceimprovements) des json parser und serializers.
